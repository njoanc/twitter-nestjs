## Description

Build A Twitter Clone With NestJS, Prisma And React.

## Installation

```bash
$ npm install
```

## Requirements

```
$ Web APIs and HTTP
$ NodeJS and npm
$ Typescript and JavaScript
$ MySQL
$ React
```

## Setup

1. npm install -g @nestjs/cli
2. mkdir twitter && cd twitter
3. nest new twitter-api && cd twitter-api
4. npm run start:dev

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
